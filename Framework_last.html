Kick start

CSS declaration

The Angrybee Web framework is mainly based upon classes starting always by ab-....

The Angrybee Web framework is responsive and dark mode.

We propose 1 CSS adapted to all screen sizes with adaptative CSS classes for:

  • screen < 576px
  • 577px < screen < 768px
  • 769px < screen < 992px
  • 993px < screen < 1200px
  • 1201px < screen < 1400px
  • 1401px > screen

You don't need to deal with screen sizes, classes are applied depending on the screen sizes.


Angrybee CSS declaration in HTML


<link rel="stylesheet" href="$url/angrybee-dark-pc.css">

Javascript declaration

We propose also a set of Javascript libraries, designed by Angrybee but also by other Open Source projects to enhance the power of our framework and avoid reinventing the wheel.

You just have to adapt the $url path to adapt the code sample you can copy/paste.

Javascript files declaration


                        
<header>
    ...
    <script src="$url/highlight.min.js"></script>
    <script src="$url/anchor.min.js"></script>

    <script src="$url/kute.min.js"></script>
    <script src="$url/vivus.min.js"></script>
    <script src="$url/snap.svg-min.js"></script>
    <script src="$url/angrybee.js"></script>

<script>
    //Enable syntax highlight
    hljs.highlightAll();

</script>

CSS Details

Default CSS

Angrybee has built its own CSS classes to propose a consistent library for its Dark Mode.

Angrybee

Apache 2 License

Angrybee CSS Framework.

angrybee-dark-pc.css
angrybee-dark-mobile.css

Angrybee Symbol

Apache 2 License

Angrybee CSS dedicated to symbols.

angrybee-symbols.css

Extra-CSS libraries

To extend the Angrybee framework capabilities and avoid reinventing the wheel, we have integrated other interesting CSS libraries .

Angrybee integrates the following CSS files:

Github syntax highlight

BSD-3-Clause License

CSS for syntax highlights. See highlightjs. CSS associated to highlight.min.js

github-dark.min.css

Magic

MIT License

A CSS for animations with special effects. We choose this CSS because of its specificity regarding the special effects (see Magic website and its documentation).

magic.min.css

Bootstrap

MIT License

Bootstrap framework bundled with Popper. See Bootstrap.

bootstrap.min.css

Bootstrap Icons

MIT License

Predefined Bootstrap icons. See Bootstrap icons.

bootstrap-icons.css

Hover.css

MIT License

CSS for animations dedicated to the "hover" event. See Hover.css.

hover-min.css

WickedCSS

No License

Simple effect CSS animation library, easy to use. See WickedCSS.

wickedcss.min.css

CSS included by default

You don't need to include these CSS files in your code, they are already included in the Angrybee CSS files.

CSS included by default


                        
@import url("$url/github-dark.min.css");
@import url("$url/bootstrap-icons.css");
@import url("$url/bootstrap.min.css");
@import url("$url/angrybee-symbols.css");
@import url("$url/magic.min.css");
@import url("$url/aos.css");
@import url("$url/hover-min.css");
@import url("./wickedcss/wickedcss.min.css");

Javascript Details

Some extra-CSS are delivered with Javascript so it is mandatory to get them. The Angrybee Framework is bundled with all the necessary to run these extra-libraries..

But you need to call them in your HTML page.

By default, we don't modify the source code of the CSS and JS projects we include. In some cases we add classes or we override styles.

Default integrations

AnchorJS

MIT License

Add automatically anchors to the HTML titles. See AnchorJS.

anchor.min.js

Bootstrap with CSS

MIT License

World’s most popular front-end open source toolkit designed for responsive mobile-first sites. See Bootstrap and Bootstrap icons.

bootstrap.bundle.min.js

PRELOADJS

MIT License

A JavaScript library that lets you manage and co-ordinate the loading of assets and data. See PRELOADJS.

preloadjs.min.js

AOS with CSS

MIT License

Animate On Scroll library is a simple but powerfull animation library to manage appearence of elements during scrolling. See AOS.

aos.js

KUTE.js

MIT License

Advanced animation library providing a huge number of animations such as SVG transformation, text properties, transformation matrix, etc. See KUTE.js.

kute.min.js

Vivus

MIT License

Lightweight JavaScript class (with no dependencies) that allows you to animate SVGs, giving them the appearence of being drawn. See Vivus.

vivus.min.js

Snap.svg

MIT License

Advanced JavaScript to bring life to SVGs. See Snap.svg.

snap.svg-min.js
